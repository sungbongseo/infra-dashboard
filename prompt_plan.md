## Phase 2: 교차 분석 — 영업 의사결정 인사이트

### P2-1. 거래처 수익성 x 미수금 리스크 매트릭스 (거래처 단위)
- 교차: orgCustomerProfit(303) + receivableAging → 거래처코드 매칭
- 4사분면: 고수익-저리스크(유지) / 고수익-고리스크(수금관리) / 저수익-저리스크(마진개선) / 저수익-고리스크(거래축소)

### P2-2. 계정구분별 수익성 대시보드 (제품 vs 상품 vs 원자재)
- P1-1/P1-3에서 파싱된 계정구분 활용
- 계정구분별 매출 비중 추세, 매출원가율 차이, 조직별 제품/상품 비중

### P2-3. 수주 전환율 분석 + 파이프라인 가치 평가
- 교차: orderList(품목상태) + salesList + orgProfit(과거 마진율)
- 수주 전환율 + 미결 수주 예상 이익 추정

### P2-4. 303 거래처별 판관비 드릴다운
- orgCustomerProfit col23~61 판관비 세부 13항목 파싱 확장
- 거래처별 운반비/수수료/노무비 워터폴

### P2-5. 표준원가 vs 실적원가 차이 분석
- 200파일 표준매출원가 + 실적매출원가
- 품목별 원가 차이율, 원가 초과/절감 품목 식별

## Phase 3: 기존 탭 인사이트 강화

### P3-1. 유형별 탭 → 계정구분/품목범주 다차원 분석
- 현재 내수/수출 2분류 → 계정구분 x 품목범주 x 사업장 3차원

### P3-2. 거래처 탭(Sales) + 거래처 손익(Profitability) 연계
- 거래처 탭에 수익성+미수금 컬럼 추가

### P3-3. 품목군(34개) 기반 포트폴리오 분석
- P1-3 품목군 활용, BCG 매트릭스, 조직 x 품목군 히트맵

### P3-4. 수금지연 추정 분석 (거래처 단위)
- salesList + collectionList → 거래처별 FIFO 수금지연 추정
